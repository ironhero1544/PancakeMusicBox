# PancakeMusicBox
PancakeMusicBox는 사용자 친화적인 UI를 갖춘 HiFi 음질 재생용 로컬 파일 기반 음악 플레이어입니다. / Pancake Music Box is a local file-based music player for HiFi quality playback with a user-friendly UI.
